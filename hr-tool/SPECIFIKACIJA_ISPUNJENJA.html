1️⃣ Database Model i Tabele ✓ KOMPLETNO

Implementirani Entiteti:

  • JobCandidate - Entitet sa svim traženim atributima
  • Skill - Entitet za skills
  • Many-to-Many relacija između JobCandidate i Skill

Atributi JobCandidate:

  • id (PRIMARY KEY, AUTO_INCREMENT)
  • fullName (NOT NULL)
  • dateOfBirth (NOT NULL)
  • contactNumber (NOT NULL)
  • email (NOT NULL)
  • skills (Set<Skill> - Many-to-Many)

Atributi Skill:

  • id (PRIMARY KEY, AUTO_INCREMENT)
  • name (NOT NULL, UNIQUE)

Baza podataka:

  • H2 in-memory baza
  • Automatska kreira tabela: job_candidates, skills, job_candidate_skills
  • DDL-AUTO: update (automatska migracija)
