ZAHTEV 3: REST WEB SERVICES
═════════════════════════════════════════════════════════════════════════════════

REST Controller:
✅ JobCandidateController
   ✅ Base path: /api/candidates
   ✅ @RestController annotation
   ✅ Constructor injection (JobCandidateService)

REST Endpoints (8 Operacija):

✅ 1. ADD JOB CANDIDATE
   ✅ Method: POST
   ✅ Path: /api/candidates
   ✅ Response Status: 201 CREATED
   ✅ Request: JobCandidateRequest
   ✅ Response: JobCandidateResponse
   ✅ Validation: @Valid on request

✅ 2. UPDATE JOB CANDIDATE
   ✅ Method: PUT
   ✅ Path: /api/candidates/{id}
   ✅ Response Status: 200 OK
   ✅ Request: JobCandidateRequest
   ✅ Response: JobCandidateResponse
   ✅ Validation: @Valid on request

✅ 3. ADD SKILLS TO CANDIDATE
   ✅ Method: POST
   ✅ Path: /api/candidates/{id}/skills
   ✅ Response Status: 200 OK
   ✅ Request: List<String> (skill names)
   ✅ Response: JobCandidateResponse

✅ 4. REMOVE SKILL FROM CANDIDATE
   ✅ Method: DELETE
   ✅ Path: /api/candidates/{id}/skills/{skillName}
   ✅ Response Status: 200 OK
   ✅ Response: JobCandidateResponse

✅ 5. REMOVE CANDIDATE
   ✅ Method: DELETE
   ✅ Path: /api/candidates/{id}
   ✅ Response Status: 204 NO_CONTENT

✅ 6. GET CANDIDATE BY ID
   ✅ Method: GET
   ✅ Path: /api/candidates/{id}
   ✅ Response Status: 200 OK
   ✅ Response: JobCandidateResponse

✅ 7. SEARCH CANDIDATES BY NAME
   ✅ Method: GET
   ✅ Path: /api/candidates/search/by-name
   ✅ Request Param: name (String)
   ✅ Response Status: 200 OK
   ✅ Response: List<JobCandidateResponse>

✅ 8. SEARCH CANDIDATES BY SKILLS
   ✅ Method: GET
   ✅ Path: /api/candidates/search/by-skills
   ✅ Request Param: skills (List<String>)
   ✅ Response Status: 200 OK
   ✅ Response: List<JobCandidateResponse>

Data Transfer Objects:
✅ JobCandidateRequest (record)
   ✅ fullName: @NotBlank
   ✅ dateOfBirth: @NotNull
   ✅ contactNumber: @NotBlank
   ✅ email: @Email @NotBlank
   ✅ skills: List<String> (optional)

✅ JobCandidateResponse (record)
   ✅ id: Long
   ✅ fullName: String
   ✅ dateOfBirth: LocalDate
   ✅ contactNumber: String
   ✅ email: String
   ✅ skills: List<String> (sorted alphabetically)

Exception Handling:

   ✅ EntityNotFoundException → 404 NOT_FOUND
   ✅ IllegalArgumentException → 400 BAD_REQUEST

   ✅ Error response format: {"error": "message"}
