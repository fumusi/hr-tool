# hr-tool

This project represents a simple HR platform for managing job candidates and their skills. It includes basic CRUD operations, as well as searching candidates by name and skills.

## Technical Decisions

One of the decisions I made during development was to use the H2 in-memory database instead of PostgreSQL. The main reason for this was that I wanted to try out a new technology and simplify the setup process. Using H2 allowed me to run and test the application quickly without additional configuration, while still working with JPA and relational data concepts.

## Most Interesting / Challenging Part

The most interesting part for me was the part with the H2 because it was new technology to me.


It also gave me the opportunity to experiment with H2 and improve my backend development workflow.
